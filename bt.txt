Exercise: Library Management SystemBuild a library system that manages books and members.Classes to implementBook

Attributes: title, author, isbn, is_available (default True)
Methods:

checkout() — mark as unavailable, raise error if already checked out
return_book() — mark as available, raise error if not checked out
_str_ — readable representation


Member

Attributes: name, member_id (auto-generated), borrowed_books (list)
Methods:

borrow(book) — add book to borrowed list, raise error if member has ≥ 3 books
return_book(book) — remove from borrowed list, raise error if book not in their list
_str_ — show name + number of books borrowed


Library

Attributes: name, books (list), members (list)
Methods:

add_book(book) — add to collection
register_member(member) — add to members
checkout_book(member, isbn) — find book by ISBN, call both book.checkout() and member.borrow(book)
return_book(member, isbn) — reverse of above
search(keyword) — return list of books matching title or author (case-insensitive)
available_books() — return list of available books
_str_ — show library name + counts
